# Backend API

Struktur ini mengikuti pola Laravel agar mudah dipindahkan ke instalasi framework penuh.

## Fitur yang Sudah Disiapkan

- Migration skema inti
- Model domain
- Seeder role dan user demo
- Middleware role
- Request validation
- Controller API
- Endpoint laporan dan dashboard
- Dokumentasi endpoint

Dokumentasi endpoint ada di [../docs/api-endpoints.md](/workspace/air-bersih-management/docs/api-endpoints.md).
