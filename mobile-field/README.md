# Mobile Field App

Struktur mobile ini disiapkan untuk Android dengan React Native. Fokus role:

- Login petugas lapangan
- Daftar pelanggan lapangan
- Pencatatan meter
- Penerimaan pembayaran
- Tindak lanjut keluhan

Folder `android/` disiapkan sebagai placeholder integrasi native saat project dijalankan melalui React Native CLI di mesin development penuh.
